### Coordinate systems / handedness
Mustard Engine uses a right-handed coordinate system, with the negative Z axis pointing outward from the camera.

### Subsystems
- Audio
- Renderer
- Window (on desktop platforms)
- Script

### Asset Managers
Several asset managers are defined:  

- Sprites
- Shaders
- Fonts
- Sounds
- Music
